GAMBIT

Gambit é um software utilitário de agendamento de desligamento para Windows feito na linguagem de programação Go com a biblioteca Fyne GUI para interface gráfica, feito por Guilherme Faura. O Gambit está licenciado sob a licença Apache-2.0, leia-a com atenção.

WINDOWS SMARTSCREEN OU ANTIVÍRUS IMPEDINDO O USO

Infelizmente, pelos testes que fiz no Windows, alguns antivírus removem o .exe do Gambit ou em outros casos, ao tentar executar, o Windows SmartScreen impede seu uso com uma mensagem de segurança. Observe que não há nada malicioso sobre o Gambit, a fonte está aqui e você pode verificá-la ou compilá-la você mesmo. O problema é que o Gambit é um programa sem assinatura digital e também não está na loja da Microsoft e por isso os antivírus ou o Windows SmartScreen acham que seja malicioso sem realmente verificar se realmente seja algo malicioso. E a crítica que deixo aqui é que é muito caro no meu país pagar uma assinatura digital mensal para manter um programa sem fins lucrativos. Portanto, o Windows SmartScreen rotular qualquer coisa de malware não é coincidência na minha opinião, bom trabalho Microsoft, a cada dia mais motivos para me fazer preferir o mundo Open Source...


COMO USAR?

O uso é simples. Dada uma entrada de tempo no formato HH:MM, Gambit agendará um desligamento para esse horário calculando automaticamente a diferença em segundos entre duas datas (data programada e data atual).
