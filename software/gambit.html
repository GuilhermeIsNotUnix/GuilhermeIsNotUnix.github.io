GAMBIT

Gambit é um software utilitário de agendamento de desligamento para Windows feito na linguagem de programação Go com a biblioteca Fyne GUI para interface gráfica, feito por Guilherme Faura.

O Gambit está licenciado sob a licença Apache-2.0, leia-a com atenção.

file_downloadDownload

COMO USAR?

O uso é simples. Dada uma entrada de tempo no formato HH:MM, Gambit agendará um desligamento para esse horário calculando automaticamente a diferença em segundos entre duas datas (data programada e data atual).
